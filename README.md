# Kinopoisk-account-migrator

Инструмент для переноса данных между аккаунтами Кинопоиска: оценки, просмотренное и список «Буду смотреть».

## Что делает

- **Экспортирует** из исходного аккаунта: оценки фильмов, все просмотренные, список «Буду смотреть»
- **Импортирует** в целевой аккаунт: выставляет оценки, отмечает просмотренное, добавляет в «Буду смотреть»
- Поддерживает **resume** - если импорт прервался, при следующем запуске автоматически найдёт сохранённый прогресс и продолжит с того места где остановился
- Сохраняет подробный отчёт о результатах импорта

## Как это работает

Скрипты используют Selenium (автоматизация Chrome) - браузер открывается на экране, вы логинитесь вручную, дальше скрипт работает сам. Ручной логин намеренный: это избавляет от капчи и необходимости хранить пароли.

## Требования

- Python 3.9+
- Google Chrome

## Установка

```bash
pip install selenium beautifulsoup4
```

> Chromedriver скачивается автоматически через Selenium Manager (Selenium 4.6+). Ничего дополнительно устанавливать не нужно.

## Быстрый старт

### Полная миграция (экспорт → импорт в один запуск)

```bash
python main.py
```

Скрипт сначала попросит войти в **исходный** аккаунт (откуда переносим), затем - в **целевой** (куда переносим).

### Только экспорт

```bash
python main.py --mode export
```

Данные сохранятся в папку `kinopoisk_export_<дата>/`.

### Только импорт из готового экспорта

```bash
python main.py --mode import --input ./kinopoisk_export_20260406_082649
```

## Все флаги

```
--mode {all,export,import}     Режим: all (по умолчанию), export или import

Браузер:
  --chromedriver PATH          Путь к chromedriver (обычно не нужен)
  --chrome-binary PATH         Путь к бинарнику Chrome если не находится автоматически
  --export-profile-dir PATH    Chrome-профиль для исходного аккаунта
  --import-profile-dir PATH    Chrome-профиль для целевого аккаунта

Экспорт:
  --export-output-dir PATH     Куда сохранять файлы экспорта
  --max-pages N                Лимит страниц пагинации (default: 500)

Импорт:
  --input PATH                 Папка с экспортом или файл kinopoisk_export.json
  --import-output-dir PATH     Куда сохранять отчёт импорта
  --delay SEC                  Задержка между фильмами в секундах (default: 1.0)
  --skip-ratings               Не импортировать оценки
  --skip-watched               Не импортировать просмотренное
  --skip-watchlist             Не импортировать «Буду смотреть»
  --dry-run                    Показать сколько записей будет импортировано, без реальных действий
  --no-resume                  Начать импорт заново, игнорируя сохранённый прогресс
```

## Структура файлов экспорта

```
kinopoisk_export_<дата>/
├── kinopoisk_export.json   # Всё в одном файле (для импорта)
├── ratings.json            # Оценки
├── watched.json            # Просмотренные
├── watchlist.json          # «Буду смотреть»
├── ratings.csv             # Оценки в CSV
├── watched.csv
├── watchlist.csv
└── summary.json            # Статистика экспорта
```

## Результаты импорта

```
kinopoisk_import_<дата>/
├── import_report.json      # Подробный отчёт: статус каждого фильма
└── import_state.json       # Сохранённый прогресс для resume
```

## Ограничения

- Работает только с публичными страницами фильмов/сериалов на `kinopoisk.ru`
- Импорт «просмотренного» - best effort, наиболее нестабильная часть (зависит от вёрстки)
- Кинопоиск может менять разметку - тогда потребуется обновить селекторы
